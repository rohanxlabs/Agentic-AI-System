🧠 What is Agentic AI?

Traditional AI responds.
Agentic AI acts.

This system demonstrates how AI agents can:

- 🎯 Set goals
- 🧩 Break problems into tasks
- 🔄 Execute multi-step workflows
- 🛠️ Use tools & APIs
- 🧠 Adapt based on results

Agentic systems are defined by autonomy, planning, and tool usage, making them fundamentally different from simple chatbots.

---

🎯 Problem Statement

Most AI applications today are:

❌ Static (single prompt → response)
❌ Non-adaptive
❌ No memory or reasoning

This project explores:

✅ Autonomous decision-making
✅ Multi-step reasoning
✅ Task execution pipelines

        ┌──────────────┐
        │   User Goal   │

               ↓
        ┌──────────────┐

               ↓

        └──────┬───────┘
               ↓
        ┌──────────────┐
        │ Memory Update │
        └──────┬───────┘
               ↓
        🔁 Loop until goal achieved
